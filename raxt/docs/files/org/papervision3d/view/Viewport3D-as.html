lastRenderList

public var lastRenderList: Array

lastRenderList stores RenderableListItems (Triangles, Lines, Pixels, Particles, Fog) of everything that was rendered in the last pass.  This list is used to determine hitTests in hitTestPoint2D.

See Also

hitTestPoint2D()

interactiveSceneManager

public var interactiveSceneManager: InteractiveSceneManager

interactiveSceneManager manages the interaction between the user’s mouse and the Papervision3D scene.  This is done by checking the mouse against renderHitData. renderHitData is generated from hitTestPoint2D and passed into the interactiveSceneManager to check agains the various mouse actions.

See Also

hitTestPoint2D(), org.papervision3d.core.utils.InteractiveSceneManager.renderHitData

Parameters

viewportWidthWidth of the viewport
viewportHeightHeight of the viewport
autoScaleToStageDetermines whether the viewport should resize when the stage resizes
interactiveDetermines whether the viewport should listen for Mouse events by creating an InteractiveSceneManager
autoClippingDetermines whether DisplayObject3Ds outside the rectangle of the viewport should be rendered
autoCullingDetemines whether only the objects in front of the camera should be rendered.  In other words, if a triangle is hidden by another triangle from the camera, it will not be rendered.
